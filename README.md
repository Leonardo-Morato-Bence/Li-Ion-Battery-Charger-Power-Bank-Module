# Li-Ion-Battery-Charger-Power-Bank-Module
High-efficiency single-cell Li-Ion charger and power bank module featuring USB-C charging, battery protection and regulated 5V output.

## OVERVIEW

This project implements a compact battery management and power bank solution for a single 18650 Li-Ion cell.

The design integrates:

- Battery charging
- Battery protection
- USB-C input
- 5V USB output
- Battery level indication
- Power path management
- ESD protection
- Resettable fuse

The module is intended for portable electronics, embedded systems and prototyping applications.

-------------------------------------------------------------------------------------------------------------

FEATURES

- USB Type-C charging input
- Single-cell 18650 Li-Ion battery support
- 4.2V CC/CV charging
- Regulated 5V USB output
- Battery level LED indicators
- Push-button control
- Overcharge protection
- Overdischarge protection
- Overcurrent protection
- Short-circuit protection
- ESD protection
- Resettable battery fuse
